<objective>
Build the pure data layer that turns the committed derived files into the marker inputs the map will render: resolve a station's content-hashed filename from `manifest.json`, fetch + `decodeDerived` (from the `/derive` subpath so no Node built-ins reach the browser bundle — Risk #1), then compute the default-period per-station averages with `@betravedur/domain` (no reimplementation) into a `MarkerDatum`. This is a self-contained, fully unit-testable slice (TDD) — no map/DOM code — so the transform's correctness is proven before it drives pixels in Plan 03.

Purpose: Isolates the one genuinely-new domain-adjacent logic (period → marker datum) with the coverage-honest, "án úrkomu", and "breytileg átt" edge cases pinned by unit tests, and establishes the `MarkerDatum` contract Plan 03 consumes. Designed so a Phase-4 selector can swap the period without re-architecting.

Output: `site/src/data/{types,load,averages}.ts` + their tests, all green offline against the committed sample.
</objective>

```typescript
export interface DailyObservation {
  station: number; date: string; doy: number;      // doy = leap-folded 1..365
  t: number|null; tx: number|null; tn: number|null;
  f: number|null; fx: number|null; fg: number|null;
  dv: number|null;   // wind dir deg — null on SYNOP
  r: number|null;    // precip mm — null on AWS; missing is NEVER coerced to 0
}
export type StationType = "sj"|"sk"|"ur"|"vf";       // sj=AWS, sk=SYNOP
export interface StationMeta {
  station: number; name: string; type: StationType; owner: string;
  lat: number; lon: number; ele: number; start: number; ending: number|null;
}
export interface WindowSpec { startDoy: number; endDoy: number; }
export function expandWindow(w: WindowSpec): Set<number>;               // set of doy in-window (leap-folded)
export function groupBySeasonYear(rows: DailyObservation[], w: WindowSpec): /* per-season-year grouping */;
export function qualifyingYears(byYear, windowDays, pick: (o:DailyObservation)=>number|null): /* qualifying years */;
export function effectiveN(qualifyingYears): { n: number; sufficient: boolean };   // sufficient = n >= 3
export function scalarMeanSpeed(speeds: (number|null)[]): number | null;
export function circularMeanDirection(samples: {speed:number;dirDeg:number}[]): { dirDeg: number; resultantSpeed: number } | null;
```
