Symbol-layer collision + hybrid recipe (RESEARCH Pattern 3, verified against maplibre style spec):
```typescript
map.addSource("stations", { type:"geojson", data: featureCollection /* from MarkerDatum[] */ });
map.addLayer({
  id: "station-anchors", type: "symbol", source: "stations",
  layout: {
    "text-field": ["get","label"],          // e.g. "7°" — collision footprint proxy
    "text-size": 13, "text-font": ["Noto Sans Regular"],   // must exist in the glyphs endpoint
    "text-allow-overlap": false,            // ← native declutter: no two survivors overlap
    "text-ignore-placement": false,
    "symbol-sort-key": ["get","priority"],  // stable: major stations (low priority) win
    "text-optional": true,
  },
  paint: { "text-opacity": 0 },             // hide proxy; overlay draws the visible pill
});
map.on("idle", () => {
  const placed = map.queryRenderedFeatures({ layers:["station-anchors"] });
  renderComposite(placed);                  // DOM/canvas pills for post-collision survivors only
});
```

UI-SPEC Station Marker Anatomy (left→right, gap --space-xs): temperature °C in accent red (`7°`, Semibold 13px tabular) → wind (rotated arrow ink glyph + integer speed `5 m/s`, or "breytileg átt" label when windVariable) → precip drop glyph (omit if !hasPrecip). Muted callout (--marker-empty tokens, "ófullnægjandi gögn") when !sufficient. Min hit area 44px (padded transparent — Phase 6 click seam). Wind arrow points the direction the wind blows TOWARD; document the convention in code. Color is never the sole channel (glyph + numeral always present).
