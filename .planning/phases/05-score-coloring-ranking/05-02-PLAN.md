## Phase Goal

**As a** visitor planning an Icelandic trip, **I want to** see stations colored by their combined weather score with a legend, a ranked best-stations list, and a plain explainer, **so that** I can tell at a glance where the weather has historically been best and trust how that score was reached.

> Note (MVP_MODE): the ROADMAP `**Goal:**` line is outcome-shaped but not in `As a / I want to / so that` form. This story is derived faithfully from that goal's explicit role + capability + outcome. If a stricter source story is required, run `/gsd mvp-phase 5` to set it in ROADMAP.md.

<objective>
Make the map answer "where has it been best" visually: color each scored marker by its score (BuGn left-bar/ring + always-visible numeric score badge), and render a bottom-left legend with the color scale + a folded-in `<details>` explainer of the weights (SCORE-03). Insufficient markers stay muted/off-scale; án-úrkomu markers are fully colored + badged.

Purpose: MAP-03's coloring + legend, and SCORE-03's transparency explainer. This is the vertical slice that turns the neutral Phase-3/4 pill into a score-colored one and makes the score meaning legible in one place (the transparency differentiator).

Output: extended `buildPill` (ring/badge), `--score-*` tokens, `score.css`, `legend.ts`, `main.ts` legend mount, and the score.spec.ts criteria 1-4.
</objective>
